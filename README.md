# 🩺 Disease Symptom Checker Web App

## 📌 Overview
This project is a web-based Disease Symptom Checker that predicts possible diseases based on selected symptoms using machine learning concepts.

## 🧠 Technologies Used
- HTML
- CSS
- Machine Learning (Conceptual: Naive Bayes, Decision Tree, Random Forest)

## 🎯 Features
- Symptom selection interface
- Disease prediction result page
- Modern responsive UI design
- Easy navigation between pages

## 🖥️ Pages
- Home Page
- Symptom Checker Page
- Result Page
- About Page

## 📊 Objective
To provide early awareness of possible diseases based on user symptoms.

## 🚀 How to Run
Open `index.html` in any browser.
